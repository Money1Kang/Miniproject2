요구사항 : 외부 파일을 읽어서, Java 프로그램으로 관리할 수 있게 해주세요~

console 기반으로 동작하는 하나의 프로그램을 작성
1. 파일 입출력을 적용하여 Todo 데이터를 읽어들인다.
-> 입력, 출력
-> 입력 : todo-data-simple.txt 파일(외부 파일)을 특정 애플리케이션(ex. Java Program)에 읽어들인다(read)
-> 출력 : 특정 애플리케이션에서 특정 파일(txt, tsv, csv 등)에 데이터를 기록(wrtie)

2. 읽어들인 데이터를 Java 프로그램에서 조회한다.
3. Primitive data type(기본 타입)를 그대로 관리한다. -> 기본 타입으로 배열 생성하였음(int, LocalDate, String)
4. 배열을 활용하여 관리한다.
5. 결과 값을 출력한다.

현재 한계점
1. 객체가 아닌 기본 자료형을 그대로 활용 중이라 관리가 불편함.
